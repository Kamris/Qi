Top [unstable]

Intends to be a complete NIH HTML5 High Performance front-end for developing todays websites compatible with yesterdays computational devices (IE6/IE7 anybody?).

Thoughts behind the Top Foundation

1. box-sizing:border-box

What was long ago considered a buggy implementation by Internet Explorer has now made its way to a feature that makes it easier to create layouts. border-box: box-sizing allows you forget about subtracting borders and padding from your container in order to build your layout.

2. Layout based on inline-block

It was decided to use inline-block because it's an arguably easier thing to comprehend when compared to floats and offered features like being able to use vertical-align.
