Introduction

Qi attempts to complete the flow of styling documents through the addition of CSS4 proposals and HTML5 form element extensions.

Uses document.querySelector() in order to extend functionality to CSS by adding things like :click or offering different ways to traverse the document tree.

Example Usage

Usage is simple, but usage must be in it's own block.

selector:click { color:#000; }

Notes

Adding will invalidate your CSS code against the W3C.
