top [alpha]

A complete HTML5 high performance front-end for websites aimed towards recent screen devices with tiered fallback.

Features

  • High Performant
    • Small footprint. Under 1kb minified and gzipped.
    • Immediate paint-times.
    • Non-blocking CSS & Javascript
  • Tiered Compatibility
    • Tier 0: Enterprise browsers will still render layout in IE6/7 (Will be removed around ~version 3)
    • Tier 1: (optional) IE8 will render layout as well as functional elements.
    • Tier 2: IE9+ Gets the full experience.
