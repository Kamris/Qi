What top is

A complete HTML5 high performance front-end for developing todays websites compatible with yesterdays devices.

Features

  • High Performance
    • Tiny footprint.
    • Immediate paint-times.
    • Non-blocking CSS & Javascript.
    • jQuery Plugins onDemand
  • Tiered Compatibility
    • Tier 0: IE6/7 will render layout.
    • Tier 1: IE8 will render layout as well as additional CSS.
    • Tier 2: IE9+ and other modern browsers get the full experience.
  • Common Components
    • jQuery 1.* with a set of plugins loaded only when needed
    • normalise.css 4.x
    • FontAwesome for iconography
